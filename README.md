# Student-Management-System
Student Management System (C++)
📌 Project Description

The Student Management System is a console-based application developed using C++.
This project allows users to manage student records efficiently through a menu-driven interface.

The system supports operations such as adding, searching, updating, deleting, and displaying student records.
All student data is stored in a file so that records remain available even after the program is closed.

This project demonstrates the use of Object-Oriented Programming, file handling, and data structures in C++.

🚀 Features

Add new student

Display all students

Search student by ID

Update student details

Delete student record

Persistent file storage

🛠 Technologies Used

C++

Object Oriented Programming (OOP)

File Handling

Vector (STL)

Loops and Conditional Statements

⚙️ How It Works

The program loads student data from students.txt.

A menu is displayed to the user.

The user selects an operation.

The system performs the selected action.

Data is saved back to the file.

Example menu:

1 Add Student
2 Display Students
3 Search Student
4 Update Student
5 Delete Student
6 Exit
